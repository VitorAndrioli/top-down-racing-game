Detailed Description

Template class for every collidable object in the game.

Implements basic mechanics and integration methods to move objects, as well as broad collision tests, collision response and impulse application. Declares virtual methods for detecting collision, allowing polymorphic calls.

Author
Vitor Augusto Andrioli
