Detailed Description

Circular collidables to be used in the game.

Implements collision detection and texture allocation for circular objects. Can be used to turn background figures into immovable collidables. A circle instance is not drawn to the screen.

Author
Vitor Augusto Andrioli
Version
1.0

Constructor initializing position, radius and restitution coefficient.

Initializes position, radius and restitution coefficient with paramenters. The restitution coefficient can be omitted and, in this case, it will be set to 1 (fully elastic collision).

Parameters
fPosX,fPosYCoordinates for the position vector.
fRadiusRadius of the object.
fRestitudion(Optional) Restitution coefficient of object. If omitted, it will be set to 1.
