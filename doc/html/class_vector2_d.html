double Vector2D< G >::crossProduct ( Vector2D< G > *  pOtherVector)

Calculates cross product with another vector.

There is no cross product defined for 2D vectors. We can, however, make some abstractions: Cross product between two vectors is a scalar, that represents the magnitude in the non existent Z axis (were these 3D vectors).
